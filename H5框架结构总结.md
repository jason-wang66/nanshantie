## 三、页面结构（5 大模块）

### 1. 开篇动画（introPage）
- **墨滴下落**：SVG 墨滴从顶部坠落动画（dropFall 1.2s）
- **墨迹溅散**：径向渐变模拟溅墨效果
- **书法笔画**：SVG 路径逐笔书写"一→二→三"（stroke-dasharray 动画）
- **品牌文字**："三木不为森"逐字显现 → "南山帖"标题 → "南山有四季，四季有十二客"竖排诗句
- **入口交互**：输入山客名字 → 点击"进入南山"按钮
- **跳过按钮**：右上角 skip 按钮
- 整个动画结束后 fadeOut 消失（1.5s），露出主内容

### 2. 一木（yimuPage）—— 盲选茶运
- 标题"一木" + 副标题"缘起一木，盲选今日茶运"
- **山客网格**（guestsGrid）：15 款茶品的山客形象卡片，点击触发圣杯仪式
- **圣杯弹窗**（shengbeiDialog）：写实圣杯动画，掷杯决定是否得签
  - 掷杯动画：两个杯体翻转，随机结果（圣杯/阴杯/笑杯）
  - 圣杯 → 进入签文结果页
  - 其他 → 可重掷
- **签文结果页**（resultPage）：
  - 日期 + 山客名
  - 山客名/茶品名 + 星级
  - 宜/忌 + 签语 + 解读
  - 茶品图片 + 故事文字
  - 操作按钮：再选一签 / 分享茶运（Canvas 生成分享图）/ 结缘此茶

### 3. 二林（erlinPage）—— 产品展示
- 标题"二林" + 副标题"博览 · 洞察 · 产品"
- **旋转轮盘**（rotationWheel）：
  - 中心"南山"二字
  - 15 款茶品按季节排列成圆环
  - 支持左右滑动旋转 + 点击查看详情
- **产品详情弹窗**（productModal）：
  - 产品名 + 茶品类名
  - 山客形象图 + 故事文案
  - 产品大图
  - 风味笔记 + 藏养建议 + 等级评语
  - 产品标签
  - 结缘此茶按钮

### 4. 三森（sanlinPage）—— 品牌故事
- 沉浸式逐行显现的品牌故事文本
- 6 行文字依次淡入：
  1. "莫离在屋前煮水。"
  2. "水开了，她把水倒进碗里，热气升起来，模糊了她的脸。"
  3. "有人推门进来，问：'今日喝什么？'"
  4. "她没有回答，只是指了指窗外。"
  5. "窗外，南山正青。"
  6. "南山有四季，四季有十二客，而你，是第十三位。"
- 底部落款"三木不为森 · 南山帖"
- 关闭按钮返回

### 5. 辅助模块
- **底部导航**（navBar）：一木 / 二林 / 三森，三个 tab 切换
- **微信二维码弹窗**（wechatModal）：结缘此茶时弹出客服微信码
- **私人定制表单**（customForm）：私人茶礼定制入口
- **企业合作表单**（corporateForm）：商务合作入口
- **Toast 提示**：轻量操作反馈

## 四、数据架构

### 产品数据（PRODUCTS 数组）
15 款茶品，每款包含：
```javascript
{
  id: "tongmeng",        // 唯一标识
  name: "童濛",          // 山客名（2字）
  tea: "白毫银针",       // 茶品名
  season: "春",          // 所属季节
  category: "白茶",      // 茶类
  story: "...",          // 山客故事
  flavor: "...",         // 风味笔记
  brewing: "...",        // 冲泡参数
  value: "...",          // 藏养建议
  grade: "...",          // 等级评语
  tags: [...],           // 产品标签
  fortunes: [            // 6 组签文数据
    {
      stars: 1-5,        // 运势星级
      yi: [...],         // 宜
      ji: [...],         // 忌
      qianyu: "...",     // 签语
      interpretation: "..."  // 解读
    }
  ]
}
```

### 季节分组
- 春：童濛（白毫银针）、浮岚（高山绿茶）
- 夏：不器（白牡丹）
- 秋：丹桂（桂花红茶）、桂兰（肉桂）、天香（大红袍）、听枞（老枞）
- 冬：履霜（工夫红茶）、漱雪（茉莉绿茶）、漱月（茉莉红茶）
- 四时：莫离（茉莉银针）、半见（陈皮白茶）
- 传说：步止（12年白茶饼）、无羁（17年白茶饼）、南山（21马年茶饼）

## 五、核心交互逻辑

### 开篇动画流程
1. 墨滴下落 → 溅散
2. "一"笔画书写 → "二"笔画书写 → "三"笔画书写
3. "三木不为森"逐字显现
4. "南山帖"标题 + 诗句出现
5. 输入名字 → 进入一木页

### 茶运抽签流程
1. 点击山客卡片
2. 弹出圣杯弹窗 → 掷杯
3. 圣杯结果 → 随机选取该茶品的一组签文
4. 展示签文结果页（日期/星运/宜忌/签语/解读）
5. 分享 → Canvas 绘制分享海报图

### 旋转轮盘交互
- touch 事件驱动旋转
- 惯性滚动 + 吸附
- 点击卡片弹出产品详情

### 季节粒子系统
- 根据当前季节生成不同粒子效果（春花瓣/夏萤火/秋落叶/冬雪花）
- Canvas 渲染，覆盖在页面背景层

---

## 六、分享功能

Canvas 绘制分享图：
- 签文内容渲染为图片
- 使用 roundRect 绘制圆角卡片
- wrapText 处理中文换行
- 生成后调用 Web Share API 或下载

## 七、技术要点

| 项目 | 方案 |
|------|------|
| 框架 | 原生 HTML/CSS/JS，零依赖 |
| 动画 | CSS animation + JS 控制 |
| SVG 书法 | stroke-dasharray + dashoffset 动画 |
| 触摸交互 | touchstart/move/end 事件 |
| 粒子效果 | Canvas 2D |
| 分享图 | Canvas 绘制 + Web Share API |
| 微信码 | base64 内联图片 |
| 部署 | GitHub Pages |
| 缓存规避 | data-version 属性 + 文件名版本号 |
